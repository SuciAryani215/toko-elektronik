# Toko Elektronik

Project berbasis Laravel untuk mengelola data barang (CRUD).

## Fitur
- Tambah data barang
- Edit data barang
- Hapus data barang
- Menampilkan daftar barang

## Teknologi
- PHP (Laravel)
- MySQL
- Bootstrap (jika digunakan)

## Cara Menjalankan
1. Clone repository ini
2. Jalankan perintah:
   composer install
3. Copy file .env:
   cp .env.example .env
4. Generate key:
   php artisan key:generate
5. Atur database di file .env
6. Jalankan migrasi:
   php artisan migrate
7. Jalankan server:
   php artisan serve
